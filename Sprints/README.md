# PairBudget — Sprint Roadmap

## Overview

5 sprints covering the full improvement set. Sprints 1 and 2 are foundational and must run first.
Sprints 3, 4, and 5 are independent of each other and can run in any order after Sprint 2 is done.

```
Sprint 1 → Sprint 2 → Sprint 3 (Mobile UX)
                    → Sprint 4 (Profile)
                    → Sprint 5 (Cleanup)
```
